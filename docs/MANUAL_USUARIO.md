# Manual de Usuario

## Objetivo

NotasFlow permite guardar apuntes, editarlos, clasificarlos con etiquetas y mostrar una integracion base con metodos de pago.

## Inicio rapido

1. Ejecuta el backend PHP desde la carpeta `backend`.
2. Ejecuta el frontend desde la carpeta `frontend`.
3. Abre el navegador en `http://localhost:5173`.

## Uso del modulo de notas

1. En el formulario principal escribe el titulo de la nota.
2. Agrega el contenido o apunte.
3. Si quieres, escribe etiquetas separadas por comas.
4. Marca la casilla de nota fijada si es importante.
5. Pulsa `Crear nota`.

## Edicion y eliminacion

1. En la lista de notas pulsa `Editar` para cargar una nota en el formulario.
2. Cambia la informacion y pulsa `Actualizar`.
3. Para borrar una nota, pulsa `Eliminar` y confirma la accion.

## Busqueda

Usa el campo de busqueda para filtrar por titulo, contenido o etiquetas.

## Modulo de pagos

1. Revisa las tarjetas con los metodos disponibles.
2. En el formulario de checkout demo selecciona el metodo.
3. Elige un plan.
4. Escribe nombre y correo.
5. Pulsa `Crear checkout demo`.

## Alcance actual

- El sistema de notas es funcional.
- El modulo de pagos esta en modo demostracion.
- La URL publica se puede agregar al final del proyecto cuando se despliegue.
